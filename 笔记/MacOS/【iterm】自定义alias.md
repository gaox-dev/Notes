# **【iterm】自定义alias**

```sh
# 修改.bash_profile 文件
vim ~/.bash_profile 

# 添加新的alias myssh1
alias myssh1='ssh root@11.22.333.4'

# 编辑后更新
source ~/.bash_profile 

# 测试命令生效
myssh1 

```



## **【iterm】 快捷键**

查找: Command+f

自动完成: Command +;

命令历史: Command +Shift+H

命令回放: Command +Alt+B

iTerm2 Exposé: Cmd+ Alt+E

光标移动到行首: Ctrl+a

光标移动到行末: Ctrl+e

前移一个字符: Ctrl+f

后退一个字符: Ctrl+b

清屏: Ctrl+l

显示历史命令: Ctrl+p

倒转查找: Ctrl+r

删除当前字符: Ctrl+d

删除前一字符: Ctrl+h

删除光标之前的字符: Ctrl+w

删除光标之后整行: Ctrl+k

最大化选中的pane：Command + Shift + Enter
